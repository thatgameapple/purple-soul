# purple-soul

A minimalist terminal writing tool for geeks.

## Install

```bash
pipx install purple-soul
```

## Run

```bash
purple-soul
```

## Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+S` | Save |
| `Ctrl+N` | New file |
| `Ctrl+L` | File list + tags |
| `Ctrl+F` | Search |
| `Ctrl+E` | Copy all to clipboard |
| `Ctrl+Q` | Quit |

## Tags

Write `#tag` or `#parent/child` in your text to organize files.
